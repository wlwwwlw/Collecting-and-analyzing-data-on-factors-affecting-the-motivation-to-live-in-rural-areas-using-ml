# Сбор и анализ данных о факторах, влияющих на мотивацию проживания на сельских территориях, методами машинного обучения

 Работа посвящена анализу факторов, влияющих на миграционные процессы сельского населения Российской Федерации, с использованием официальной статистики, методов анализа данных и алгоритмов машинного обучения.

## Цель и задачи исследования

Целью исследования является выявление ключевых факторов условий жизни, связанных с направлением миграционной динамики сельского населения, а также построение модели классификации, позволяющей различать регионы с положительным и отрицательным миграционным приростом сельского населения.

Для достижения цели в работе были решены следующие задачи:

- сбор и систематизация официальных статистических данных Росстата;
- формирование набора показателей условий жизни городского и сельского населения;
- расчёт и анализ миграционных показателей;
- выявление значимых факторов с использованием корреляционного анализа и модели Random Forest;
- построение и сравнение моделей классификации миграционных процессов сельского населения;

## Источники данных

В работе использовались открытые официальные данные Росстата за 2024 год, включая:

- показатели условий жизни населения;
- данные о миграционных потоках городского и сельского населения;
- статистику социально-экономического и инфраструктурного развития территорий.

Данные собирались вручную из разрозненных таблиц Росстата и приводились к единому формату (xlsx) для последующей обработки в Python.

Дополнительно в рамках исследования был проведён социологический опрос жителей Княгининского района Нижегородской области (124 респондента), направленный на выявление субъективных оценок условий проживания и миграционных намерений.

## Структура данных

В ходе работы были сформированы следующие основные таблицы:

- city.xlsx — показатели условий жизни городского населения (6 категорий по численности, 98 факторов);
- village.xlsx — показатели условий жизни сельского населения (4 категории по численности, 98 факторов);
- citydem.xlsx — агрегированные миграционные итоги городского населения;
- villagedem.xlsx — агрегированные миграционные итоги сельского населения;
- villageregion.xlsx — региональный набор из 16 ключевых факторов условий жизни сельского населения;
- migvill.xlsx — миграционный прирост сельского населения по субъектам РФ;
- ans.xlsx — данные опроса.

## Используемые методы

В исследовании применялись:

- разведочный анализ данных;
- корреляционный анализ;
- оценка важности признаков с использованием Random Forest;
- методы машинного обучения для классификации регионов по направлению миграционной динамики.

Для задачи классификации были протестированы следующие алгоритмы:

- Logistic Regression (L2-регуляризация);
- Gaussian Naive Bayes;
- K-Nearest Neighbors (KNN);
- Support Vector Classifier (RBF kernel);
- Random Forest (различные конфигурации);
- Extra Trees;
- Gradient Boosting;
- HistGradientBoosting.

Подбор гиперпараметров для алгоритма KNN осуществлялся с использованием стратифицированной 5-кратной кросс-валидации по метрике ROC-AUC.

## Основные результаты

- Выявлен устойчивый набор факторов условий жизни, наиболее тесно связанных с миграционными процессами сельского населения.
- Показано, что миграционный отток из сельской местности в наибольшей степени связан с доступностью медицинской помощи, состоянием жилищных условий, уровнем инженерной и цифровой инфраструктуры, а также стабильностью занятости.
- Наилучшее качество классификации регионов показала модель K-Nearest Neighbors (K=6).
- Результаты машинного обучения согласуются с выводами социологического опроса, что подтверждает интерпретируемость и практическую значимость модели.
